# 42_get_next_line

May it be a file, stdin, or even a network connection, you'll always need a way to read the content line by line.
<br>
<br>compile with:<br>

```
gcc -Wall -Wextra -Werror main.c get_next_line.c get_next_line_utils.c
```

<br>run with:<br>

```
./a.out
```
